Retrospectiva do projeto
Desafios técnicos

Durante o desenvolvimento do projeto alguns desafios foram encontrados:

Aprender a estrutura e funcionamento do Playwright.

Criar testes automatizados que interagem corretamente com os elementos da página.

Implementar testes baseados em dados utilizando arquivos CSV.

Automatizar a geração de relatórios de performance utilizando Lighthouse.

Esses desafios exigiram estudo das ferramentas e experimentação com diferentes abordagens.

Uso de Inteligência Artificial

A Inteligência Artificial foi utilizada como ferramenta de apoio ao processo de aprendizagem e desenvolvimento do projeto.

Entre os principais usos estão:

compreensão de conceitos de automação de testes

auxílio na construção de scripts de automação

apoio na análise de erros encontrados durante execução dos testes

Justificativa das ferramentas utilizadas

Foram escolhidas as seguintes ferramentas:

Playwright

Ferramenta moderna de automação de testes end-to-end que permite simular interações reais do usuário em navegadores.

Lighthouse

Ferramenta amplamente utilizada para auditoria de performance de aplicações web.

A combinação dessas ferramentas permitiu validar tanto o comportamento funcional da aplicação quanto seu desempenho.

Melhorias futuras

Caso houvesse mais tempo para evolução do projeto, algumas melhorias poderiam ser implementadas:

criação de pipeline de integração contínua

geração automática de relatórios de teste

maior cobertura de cenários de teste

inclusão de testes de API
